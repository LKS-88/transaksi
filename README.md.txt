
## Fitur Utama

1. **Manajemen Produk**
   - Tambah/edit/hapus produk
   - Kategori produk
   - Manajemen stok

2. **Transaksi Penjualan**
   - Keranjang belanja
   - Checkout dan pembayaran
   - Cetak struk

3. **Laporan**
   - Laporan penjualan
   - Laporan laba/rugi
   - Stock opname

4. **Admin**
   - Manajemen kasir
   - Backup/restore data
   - Pengaturan toko

## Teknologi

- HTML5, CSS3, JavaScript
- Enkripsi

## Cara Menjalankan

1. Clone repository
2. Buka file `frontend/index.html` di browser
3. Login dengan:
   - Admin: username `admin`, password `********`
   - Kasir: username `kasir`, password `********`
